deployArsenal

  • Parameters

This action trigger an arsenal deployment and retrieve the final status of it.

Parameters

  • alternative-config: Alternative configuration to push with the deployment
  • retry-delay: Time between try of getting the deployment final status
  • retry-max: Maximum number of retry to get the deployment status
  • token: Deployment token
  • version: The version to deploy
